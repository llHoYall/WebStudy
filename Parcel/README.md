# PARCEL

Blazing fast, zero configuration web application bundler

## Start

### # Make Project

> $ yarn init

### # Install (parcel, react, babel)

> $ yarn global add parcel

### # Make App

- index.html

### # Build Project

> $ parcel index.html

## with JavaScript

### # Install (react, babel)

> $ yarn add react react-dom
>
> $ yarn add babel-preset-env babel-preset-react --dev

### # Update App

- App.js
- index.js
- index.html
- .babelrc

## with CSS

### # Install (sass, post CSS modules, autoprefixer)

> $ yarn global add node-sass
>
> $ yarn add postcss-modules
>
> $ yarn add autoprefixer

### # Update App

- styles.scss
- App.js
- .postcssrc

## with typography.js

### # Install (typography)

> $ yarn add typography

### # Update App

- typography.js
- index.js
- App.js

## Build Production

### # Update App

- package.json

### # Build

> $ yarn build
